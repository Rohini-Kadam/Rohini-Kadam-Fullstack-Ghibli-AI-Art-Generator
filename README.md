 Ghibli AI Art Generator

A full-stack AI-powered application that transforms text prompts and images into beautiful Ghibli-style artwork.

# Features

- Text-to-Image Generation
- Image-to-Image Conversion
- Fast and Responsive UI
- Secure Spring Boot REST APIs
- AI-Powered Art Generation

# Tech Stack

# Frontend

- JavaScript
- HTML/CSS
- Axios

# Backend
- Java
- Spring Boot
- Maven
